Git Flow

[] branch main : projeto está aqui
        [] branch hotfix : correções imediatas após lançamento (temporária e possui versão 1.0.0-1 Tag)
    [] branch dev (desenvolvimento) : teste e homologação
        [] branch feature (funcionalidades) : temporária até ser mesclada ao desenvolvimento
        [] branch release (lançamento) : temporária versão 1.0.0 Tag (exemplo)
